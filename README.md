# selenium-pytest-ui-tests
# QA Automation Project: UI Testing with Selenium + Pytest

## 📌 Описание проекта

Данный проект представляет собой набор автоматизированных UI-тестов, написанных с использованием Selenium WebDriver и pytest.

Тесты реализованы в рамках практического обучения тестированию программного обеспечения и направлены на проверку пользовательских сценариев веб-приложения.

Тестируемая страница:
https://techbeamers.com/selenium-practice-test-page/

---

## 🎯 Цель проекта

- Практика автоматизации тестирования UI
- Освоение Selenium WebDriver
- Работа с pytest (фикстуры, параметризация)
- Проверка поведения веб-форм и элементов интерфейса
- Обработка негативных и позитивных сценариев

---

## 🛠️ Используемые технологии

- Python
- Selenium WebDriver
- Pytest
- WebDriverWait (Explicit Waits)
- ActionChains
- Select (работа с dropdown)
- Git

## 🧪 Реализованные тесты

### ✔ 1. Работа с UI элементами
- Навигация по странице
- Взаимодействие с меню и элементами интерфейса
- Использование ActionChains

---

### ✔ 2. Проверка отсутствия дублирования результата
- Запуск действия дважды
- Проверка количества элементов результата
- Валидация отсутствия дублей

---

### ✔ 3. Валидация формы (параметризованный тест)
- Проверка некорректных email:
  - invalid-email
  - test@
  - user@.com
  - user@domain.
  - пустое значение
- Проверка, что форма НЕ отправляется
- Обработка исключений (TimeoutException)

---

### ✔ 4. Позитивный сценарий
- Заполнение формы корректными данными
- Отправка формы
- Проверка успешного результата

---

## ⚙️ Особенности реализации

- Используются фикстуры pytest для управления браузером
- Применяются явные ожидания (WebDriverWait)
- Реализована параметризация тестов
- Используются разные способы взаимодействия с элементами:
  - click()
  - JavaScript click
  - ActionChains
- Обработка ошибок и исключений
